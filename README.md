# 我的专属 AI 智能体 v2.0

一个基于 Web 的本地化 AI 智能体应用，支持写周报、学知识、改代码三大工作流，具备流式输出、对话历史、知识库检索等功能。

## ✨ 功能特性

### 🎯 三大工作模式

| 模式 | 功能说明 |
|------|----------|
| 📝 写周报 | 输入工作关键词，自动生成结构完整的周报 |
| 📚 学知识 | 输入学习主题，整理结构化学习笔记 |
| 💻 改代码 | 粘贴代码片段，分析问题并给出优化方案 |

### 🛠 核心功能

- **流式输出**：实时显示 AI 回复，打字机效果
- **对话历史**：本地保存对话记录，随时回看继续
- **知识库检索**：上传文档，AI 基于知识库内容回答
- **多模型支持**：兼容 DeepSeek、通义千问、智谱、月之暗面、OpenAI、Ollama 等
- **深色模式**：支持深浅主题切换
- **多种主题色**：蓝、紫、绿、橙四套配色
- **代码高亮**：Markdown 代码块语法高亮
- **一键导出**：支持导出 Markdown / 纯文本格式
- **代码插入**：快捷插入代码块，支持文件上传

## 🚀 快速开始

### 环境要求

- Python 3.8+
- 任意 OpenAI 兼容 API 的 API Key

### 安装运行

1. **克隆项目**
```bash
git clone https://github.com/Futao-Augenstern/agent-web.git
cd agent-web
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **启动服务**
   - Windows：双击 `启动.bat`
   - 命令行：`python agent.py`

4. **打开浏览器**访问 `http://127.0.0.1:8765`

5. **配置 API Key**：在左侧栏选择模型供应商并填入 API Key

## 📖 使用说明

### 写周报

1. 左侧选择「写周报」模式
2. 输入本周工作关键词（可输入多个，每行一个或用逗号分隔）
3. 点击发送，AI 自动生成包含以下结构的周报：
   - 本周工作完成情况
   - 遇到的问题与解决方案
   - 下周工作计划
   - 其他事项

### 学知识

1. 左侧选择「学知识」模式
2. 输入想学习的主题（如「React Hooks」「Python 装饰器」）
3. AI 会整理结构化的学习笔记：
   - 核心概念
   - 关键知识点
   - 实践案例
   - 学习资源推荐

### 改代码

1. 左侧选择「改代码」模式
2. 粘贴需要优化的代码（或点击「插入代码」按钮）
3. 描述遇到的问题或优化需求
4. AI 会分析并给出：
   - 问题诊断
   - 优化建议
   - 改进后的代码

### 知识库

1. 点击左侧「知识库」按钮
2. 输入文档标题和内容
3. 点击「添加文档」
4. 之后的对话中 AI 会自动检索相关内容

## 🔧 配置说明

### 支持的模型供应商

| 供应商 | 名称 | 默认模型 |
|--------|------|----------|
| Agnes | agnes | agnes-2.0-flash |
| DeepSeek | deepseek | deepseek-chat |
| 通义千问 | qwen | qwen-plus |
| 智谱 AI | glm | glm-4 |
| 月之暗面 | moonshot | moonshot-v1-8k |
| OpenAI | openai | gpt-4o |
| Ollama（本地） | ollama | qwen2.5:7b |

### 环境变量

可选配置 `.env` 文件：

```env
# 默认端口（可选，默认 8765）
PORT=8765

# 默认模型（可选）
DEFAULT_PROVIDER=deepseek
```

## 🧠 技术栈

**后端：**
- Python 标准库（无需额外框架）
- HTTP 流式响应（SSE）
- 本地文件存储

**前端：**
- 原生 HTML / CSS / JavaScript（零依赖）
- highlight.js（代码高亮）
- 响应式设计，支持移动端
